Qtile Configuration Cheat Sheet

A summary of all keybindings, chords, and mouse actions defined in config.py.

Primary Modifier (mod): Super/Windows Key

Alternate Modifier (mod1): Alt Key

Window Navigation & Manipulation

Keybinding Action
mod + hMove focus to left window
mod + lMove focus to right window
mod + jMove focus down
mod + kMove focus up
mod + spaceMove window focus to next window in stack
mod + shift + hMove focused window to the left
mod + shift + lMove focused window to the right
mod + shift + jMove focused window down in stack
mod + shift + kMove focused window up in stack

Window Sizing

Keybinding Action
mod + control + hGrow focused window to the left
mod + control + lGrow focused window to the right
mod + control + jGrow focused window down
mod + control + kGrow focused window up
mod + nReset all window sizes to default

Application Launchers

Keybinding Action
mod + ReturnLaunch default terminal (kitty)
mod + rSpawn a command using Qtile's prompt widget

Window & Qtile Management

Keybinding Action
mod + wKill focused window
mod + fToggle fullscreen for focused window
mod + tToggle floating state for focused window
mod + shift + ReturnToggle between split/unsplit sides of stack (Columns layout)
mod + control + rReload Qtile configuration
mod + control + qShutdown Qtile
mod + control + shift + zToggle window gaps

Key Chords

Rofi Launcher (mod + tab)

TabRofi: Show applications (drun)
wRofi: Show open windows
qRofi: Show run command prompt
fLaunch XFE file explorer
;Launch kitty
tLaunch Codium text editor
eOpen edtr3.html

Brightness Control (mod1 + j)

kIncrease screen brightness by 20%
jDecrease screen brightness by 20%

Volume Control (mod1 + u)

iIncrease volume by 10%
uDecrease volume by 10%

Bluetooth Control (mod1 + b)

sStart scanning for devices (interactive)
SStop scanning for devices
pPair with a device (interactive)
cConnect to a device (interactive)
dDisconnect from a device (interactive)
lList paired devices (interactive)
tTrust a device (interactive)
fForget a device (interactive)
oToggle Bluetooth power

Groups (Workspaces)

Keybinding Action
mod + [1-9]Switch to group [1-9]
mod + shift + [1-9]Move focused window to group [1-9] & follow

Virtual Terminal (VT) Switching

Keybinding Action
control + mod1 + f[1-7]Switch to Virtual Terminal [1-7]

Mouse Bindings

Binding Action
mod + Left-click DragMove floating window
mod + Right-click DragResize floating window
mod + Middle-clickBring window to front
